# RISC-V-32I FDEMW-CPU

An attempt to build a five-stage in-order RISC-V-32I CPU in Systemverilog.

ISA Reference: `https://docs.riscv.org/reference/isa/unpriv/rv32.html`

## Reference Books:

1) Patterson, D. A., & Hennessy, J. L. (2020). Computer Organization and Design RISC-V Edition: The Hardware Software Interface (2nd ed.). Morgan Kaufmann.

2) Hennessy & Patterson, Computer Architecture: A Quantitative Approach, 5th ed., 2011.
